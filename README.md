# Fisching-Teste

Teste para o jogo principal Fisching.

## Minigame de pesca

Este repositório agora inclui um minigame em `pygame` com:

- peixe representado por um traço perpendicular à barra de controle;
- movimento aleatório baseado em resiliência (mais resiliência = menos movimento);
- barra branca controlada pelo mouse (pressiona = esquerda, solta = direita);
- progresso acumulado por pelo menos 6 segundos dentro da barra;
- modificadores de progresso do peixe e controle da vara.

### Como rodar

```bash
python minigame.py
```

Requer `pygame` instalado no ambiente.
